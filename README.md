# Alpha Iota Chapter Website

Official website of the Alpha Iota Chapter of Kappa Alpha Psi Fraternity, Inc. at Morgan State University.

## About

Founded May 29, 1931, the Alpha Iota Chapter continues the mission of achievement in every field of human endeavor.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm i

# Start development server
npm run dev
```

## Deployment

The site is deployed at: https://ainupes1931.com
